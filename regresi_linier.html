Regresi Linier Sederhana dengan Operasi Matriks#

Notebook ini berisi pengerjaan regresi linier berdasarkan data titik A sampai G, lengkap dengan rumus, tabel data, perhitungan manual menggunakan matriks, prediksi, RMSE, dan gambar grafik seperti GeoGebra.

1. Data yang Digunakan#

Variabel x digunakan sebagai input, sedangkan y adalah nilai yang ingin diprediksi.

2. Rumus Regresi Linier#

Model regresi linier sederhana:

\[ y = b_0 + b_1x \]

Keterangan:

  • \(y\) = nilai aktual atau nilai yang diprediksi

  • \(x\) = variabel input

  • \(b_0\) = intercept

  • \(b_1\) = koefisien regresi

Untuk mencari nilai \(b_0\) dan \(b_1\), digunakan rumus operasi matriks:

\[ \hat{\beta} = (X^TX)^{-1}X^TY \]

Dengan:

\[\begin{split} \hat{\beta} = \begin{bmatrix} b_0 \\ b_1 \end{bmatrix} \end{split}\]

3. Membentuk Matriks X dan Y#

Karena model memiliki intercept \(b_0\), maka matriks \(X\) diberi kolom tambahan berisi angka 1.

\[\begin{split} X = \begin{bmatrix} 1 & 2 \\ 1 & 4 \\ 1 & 5 \\ 1 & 3 \\ 1 & 3 \\ 1 & 4 \\ 1 & 5 \end{bmatrix} \quad Y = \begin{bmatrix} 2 \\ 3 \\ 5 \\ 4 \\ 3 \\ 5 \\ 6 \end{bmatrix} \end{split}\]
# Membuat matriks X dan Y
X = np.column_stack((np.ones(len(x)), x))
Y = y.reshape(-1, 1)

print("Matriks X:")
print(X)

print("\nMatriks Y:")
print(Y)

4. Menghitung Koefisien Regresi dengan Rumus Matriks#

Langkah perhitungan:

\[ \hat{\beta} = (X^TX)^{-1}X^TY \]

Tahapannya:

  1. Hitung \(X^T\)

  2. Hitung \(X^TX\)

  3. Hitung \((X^TX)^{-1}\)

  4. Hitung \(X^TY\)

  5. Kalikan semuanya untuk mendapatkan \(\hat{\beta}\)

# Transpose X
XT = X.T

# X transpose dikali X
XTX = XT @ X

# Invers dari XTX
XTX_inv = np.linalg.inv(XTX)

# X transpose dikali Y
XTY = XT @ Y

# Beta = (X^T X)^-1 X^T Y
beta = XTX_inv @ XTY

b0 = beta[0, 0]
b1 = beta[1, 0]

print("X Transpose:")
print(XT)

print("\nX^T X:")
print(XTX)

print("\n(X^T X)^-1:")
print(XTX_inv)

print("\nX^T Y:")
print(XTY)

print("\nHasil beta:")
print(beta)

print(f"\nIntercept b0 = {b0:.4f}")
print(f"Koefisien b1 = {b1:.4f}")
print(f"Persamaan regresi: y = {b0:.4f} + {b1:.4f}x")

5. Hasil Persamaan Regresi#

Dari hasil perhitungan, diperoleh:

\[ b_0 = 0 \]
\[ b_1 = 1.0769 \]

Jadi persamaan regresinya adalah:

\[ y = 0 + 1.0769x \]

atau dapat ditulis:

\[ y = 1.0769x \]

Artinya, setiap nilai \(x\) naik 1, maka nilai \(y\) diprediksi naik sekitar 1.0769.

6. Menghitung Prediksi dan Error#

Setelah mendapatkan persamaan regresi, kita menghitung nilai prediksi:

\[ \hat{y} = b_0 + b_1x \]

Error dihitung dengan rumus:

\[ Error = y - \hat{y} \]
# Menghitung prediksi y
Y_pred = b0 + b1 * x

# Menghitung error
error = y - Y_pred

hasil = pd.DataFrame({

7. Menghitung RMSE#

RMSE digunakan untuk mengetahui besar rata-rata kesalahan prediksi model.

Rumus RMSE:

\[ RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y}_i)^2} \]

Dalam bentuk matriks:

\[ RMSE = \sqrt{\frac{1}{n}(Y-\hat{Y})^T(Y-\hat{Y})} \]

Semakin kecil nilai RMSE, maka hasil prediksi model semakin baik.

# Menghitung RMSE
rmse = np.sqrt(np.mean(error**2))

print(f"RMSE = {rmse:.4f}")

RMSE = 0.6954

9. Kesimpulan#

Berdasarkan data titik A sampai G, diperoleh persamaan regresi linier:

\[ y = 1.0769x \]

Nilai koefisien regresi bernilai positif, sehingga hubungan antara \(x\) dan \(y\) juga positif. Artinya, ketika nilai \(x\) meningkat, nilai \(y\) juga cenderung meningkat.

Model regresi ini dibuat dengan rumus matriks:

\[ \hat{\beta} = (X^TX)^{-1}X^TY \]

Hasil prediksi kemudian dibandingkan dengan data asli menggunakan error dan RMSE.
